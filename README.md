# PCConfig - ჩემი პროექტი

ეს არის ჩემი გაკეთებული საიტი ფითონზე და ჯანგოზე (Django). პროექტი მომხმარებელს ეხმარება კომპიუტერის ნაწილების (CPU, GPU, RAM და ა.შ.) შერჩევაში და სწორი კონფიგურაციის აწყობაში.

## რა გამოვიყენე (ტექნოლოგიები):
* Python
* Django
* HTML და CSS დიზაინისთვის
* SQLite (ბაზისთვის)

## როგორ ირთვება:
1. ჩართეთ ვირტუალური გარემო
2. cd pcconfig
3. python manage.py runserver
